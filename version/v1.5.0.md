# v1.5.0

Keeper 在停更了几个版本之后，又迎来了新功能————**命令模板**。这个版本主要是配合 Keeper 更新。

## PQL

1. 新的变量类型 **调度作业变量**。
2. 新的数据类型 **哈希列表 HASHSET**。
3. Voyager 模板引擎已支持 **逻辑前置** 和 **Json 格式传递参数**。
4. 嵌入式 PQL 现在支持任意类型的文件模板。

## Keeper

1. 命令模板加入。
2. 工作流中的命令现在全部支持嵌入式 PQL。
3. 调试专有的 PQL 变量，以`%`开头。
4. 命令参数恢复支持。


## Master

1. 新增命令模板管理页面。

## root.js

1. **TABLE 标签扩展** 加入。
2. INPUT 标签扩展新增`switch`和`checkbox`类型。
